# Banana Ripeness Detection üçå

This project detects banana ripeness using image processing and machine learning.
YOLO is used for object detection and SVM for classification.

## Features
- Banana detection using YOLO
- Ripeness classification (unripe, ripe, overripe)
- ROI-based image processing

## Tech Stack
- Python
- OpenCV
- YOLO
- Scikit-learn

## Project Structure
- banana_detection.py : main program
- test.py             : testing script
- svm_banana.pkl      : SVM model
- label_encoder.pkl   : label encoder

## Model & Dataset
YOLO model weights (`best.pt`) and dataset are not included due to file size limits.
